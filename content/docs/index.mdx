Welcome to the **ads-as-code** documentation.

ads-as-code is a Pulumi-style SDK and CLI for managing ad campaigns as code. Define campaigns in TypeScript, diff against live platform state, and apply changes.

## Quick Start

```bash
# Install
npm install -g @upspawn/ads

# Initialize a project
ads init

# Preview changes
ads plan

# Apply changes
ads apply
```

## Supported Platforms

- **Google Ads** — Search campaigns, ad groups, keywords, RSAs, extensions
- **Meta Ads** — Traffic, conversions, leads, catalog sales campaigns
